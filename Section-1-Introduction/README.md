### Section 1 - Introduction

##### Install Docker.

```shell
curl -fsSL https://get.docker.com -o get-docker.sh

sudo sh get-docker.sh

sudo docker version
```

##### Test docker installation

```shell
docker run docker/whalesay cowsay Hello-World!
```
